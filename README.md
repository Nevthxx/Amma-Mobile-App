අම්මා (Amma) – Pregnancy Support App 🇱🇰
අම්මා (Amma) is a mobile application developed to support pregnant mothers in Sri Lanka with a simple, culturally relevant, and trustworthy digital companion.
While many global and local pregnancy apps exist, most lack alignment with Sri Lanka’s national healthcare practices and provide only basic information. Amma bridges this gap by offering features built around local healthcare systems and guidance from the Sri Lanka College of Obstetricians and Gynecologists (SLCOG) and the Family Health Bureau (FHB).

✨ Key Features
🧍‍♀️ User registration and login
⏰ Appointment reminders for MOH clinics, scans, and lab tests
📅 Pregnancy progress tracker – calculates current week and day of pregnancy
🩺 Structured antenatal record – allows mothers to record health data, test results, medications, and track complications
👶 Baby vaccination planner – reminders based on Sri Lanka’s National Immunization Program
📚 Information hub – provides localized content on pregnancy care, nutrition, mental health, and well-being

🛠️ Tech Stack
Android Studio (Java/Kotlin)
Firebase – Realtime Database & Authentication
Material UI Components

🎯 Motivation
This project was developed for the Mobile Application Development course as a solution to empower Sri Lankan mothers with digital tools for safer, more informed, and engaged maternity care.
